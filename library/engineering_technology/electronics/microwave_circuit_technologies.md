# Substrates and technologies

Substrate types :

- Si
- GaAs
- InP
- GaN on SiC

Les technologies SiC permettent d'obtenir de bonnes puissances à basse fréquence. Les technologies Si ne permettent que des puissances faibles (<5W) à faibles fréquences (<20 GHz). Les technologies GaAs/InP permettent de hautes fréquences, mais avec des puissances faibles.
Le meilleur compromis actuel réside dans les technologies GaN.
